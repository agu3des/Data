## Diferentes tipos de análise

Existem quatro tipos principais de análise de dados, e cada um responde a um tipo diferente de pergunta:

- A análise descritiva pergunta: “o que aconteceu?”
- A análise preditiva pergunta: "o que pode acontecer no futuro?"
- A análise prescritiva pergunta: “O que deve ser feito a seguir?”
- A análise de diagnóstico pergunta: "Por que isso aconteceu?"


### 1. Análise descritiva
    A análise descritiva usa principalmente dados observados para identificar as principais características de um conjunto de dados. Ele se baseia apenas em dados históricos para fornecer relatórios sobre eventos passados. Esse tipo de análise também é usado para gerar relatórios ad hoc (conforme necessário) que resumem grandes quantidades de dados para responder a perguntas simples como "quanto?" ou "quantos?" Também pode ser usado para fazer perguntas mais detalhadas sobre um problema específico. A análise descritiva não é usada para fazer deduções ou previsões com base nas descobertas; é apenas um ponto de partida usado para fundamentar decisões ou preparar dados para análises adicionais.

    O processo de análise descritiva é o seguinte:

    1. Faça uma pergunta histórica que precise de resposta, como "Quanto do produto X vendemos no ano passado?"
    2. Identifique os dados necessários para responder à pergunta
    3. Colete e prepare dados
    4. Análise de dados
    5. Apresentar resultados
    6. Exemplos de análise descritiva incluem:
Resumir eventos históricos, como dados de vendas, inventário ou operações
Compreensão dos dados de engajamento, como curtidas e não curtidas ou volume de visualizações de página ao longo do tempo
Relatar tendências gerais, como crescimento da receita ou lesões de funcionários
Reunindo os resultados da pesquisa

### 2. Análise Preditiva
A análise preditiva utiliza dados em tempo real e/ou passados para fazer previsões com base em probabilidades. Também pode ser usado para deduzir dados ausentes ou estabelecer uma tendência futura prevista. A análise preditiva usa modelos de simulação e previsão para sugerir o que pode acontecer no futuro, o que pode orientar a definição de metas realistas, o planejamento eficaz, o gerenciamento das expectativas de desempenho e a prevenção de riscos. Essas informações podem capacitar executivos e gerentes a adotar uma abordagem proativa e baseada em fatos para a estratégia e a tomada de decisões.

O processo de análise preditiva é o seguinte:

1. Faça uma pergunta inovadora, como "seremos capazes de prever a quantidade de produto X que venderemos no próximo ano?"
2. Colete e prepare dados
3. Desenvolva modelos de análise preditiva
4. Aplique modelos aos dados preparados
5. Analisar modelos e apresentar resultados
6. Exemplos de análise preditiva incluem:

Previsão do comportamento do cliente, padrões de compra e identificação de tendências de vendas
Prever as preferências do cliente e recomendar produtos aos clientes com base em compras anteriores e histórico de pesquisa
Prever a probabilidade de um determinado cliente comprar outro produto ou abandonar a loja sem comprar
Identificação de possíveis violações de segurança que exigem investigação adicional
Previsão das necessidades de pessoal e recursos

### 3. Análise prescritiva
A análise prescritiva se baseia na análise descritiva e preditiva, recomendando cursos de ação que colherão os maiores benefícios para a empresa. Em resumo, a análise prescritiva informa o que deve ser feito em determinada situação. Ele ajuda executivos, gerentes e funcionários a tomar as melhores decisões com base nos dados disponíveis.

Um bom exemplo de análise prescritiva é o campo de aplicações de mapa e direção baseadas em GPS. Essas aplicações oferecem opções de rota para um destino de acordo com o volume de tráfego, as condições da estrada e a velocidade máxima. A aplicação pode recomendar a melhor rota de acordo com os objetivos definidos pelo usuário, como a distância mais curta ou o tempo mais rápido.

### 4. Análise de Diagnóstico
A análise de diagnóstico aprimora o processo de análise descritiva ao se aprofundar e tentar descobrir as causas.

O processo de análise de diagnóstico é o seguinte:

1. Identifique anomalias (inconsistências) nos conjuntos de dados
2. Colete dados relacionados às anomalias
3. Use técnicas estatísticas para descobrir relacionamentos e tendências que possam explicar as anomalias
4. Apresente as possíveis causas
Um exemplo de análise de diagnóstico é usar cancelamentos de assinatura, correlacionados com comentários e classificações do cliente, para determinar os motivos mais comuns pelos quais os usuários cancelam assinaturas. Outro exemplo seria determinar se há uma correlação entre os dados demográficos dos consumidores e seus padrões de compra em épocas específicas do ano.

## Processo de Análise de Dados
- Fazendo a pergunta - O processo de análise sempre começa com uma pergunta a ser respondida. Algumas perguntas são simples, como "qual cor de bicicleta é mais popular entre nossos compradores mais jovens?" Outras são muito complexas, como “por que certos tipos de células cancerígenas estão apresentando resistência aos tratamentos de radiação?”
- Coleta dos dados - Essa etapa envolve o processo de localização e obtenção de dados relevantes para a pergunta e, em seguida, determinar se há dados suficientes para concluir a análise.
- Exploração dos dados - Os dados vêm em muitas formas e de muitas fontes diferentes. Essa etapa envolve determinar se os dados estão completos e se contêm as informações relevantes para a análise.
- Preparação e limpeza dos dados - Essa etapa pode envolver muitas tarefas para transformar os dados em um formato apropriado para as ferramentas que serão usadas para analisar e apresentar os dados. Esse processo às vezes é chamado de "limpeza" dos dados, pois pode haver registros em branco ou erros óbvios no conjunto de dados.
- Análise dos dados - A análise é o processo de identificação de padrões, correlações e relacionamentos contidos em um conjunto ou conjuntos de dados para tirar deduções e conclusões. Muitas vezes, a análise depende de técnicas estatísticas e ferramentas de software, como planilhas e aplicativos de visualização.
- Apresentando os resultados - Essa é geralmente a última etapa para os analistas de dados. É o processo de comunicação dos resultados aos tomadores de decisão. Isso pode ser feito na forma de um relatório, representações gráficas ou uma combinação de ambos. Às vezes, o analista de dados também é solicitado a recomendar ações com base nos resultados.
